Here are some things you can do with elementor plugin.

Add UI library 
# Add new elementor widget to your plugin
nx g @betrue/react-elementor:addWidget my-cool-elementor-widget --plugin=shooting-range

# Add a component
nx g @nx/react:component xyz --project=shooting-range
View dependency graph 
nx dep-graph
Run affected commands 
# see what's been affected by changes
nx affected:dep-graph

# run tests for current changes
nx affected:test

# run e2e tests for current changes
nx affected:e2e
